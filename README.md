# 📚 Student Performance & Engagement Analysis
**Tools:** Python (Pandas, Matplotlib, Seaborn) · SQL  
**Dataset:** Students Performance in Exams — 1,000 students  
**Author:** Keerthi RK | Data Analyst

---

## 🎯 Project Overview
Analyzed student engagement and performance patterns to identify at-risk students and measure the impact of structured learning programs — directly applicable to EdTech and global education institutions tracking student journeys across multiple markets.

This project mirrors real-world analytics work done to improve student outcomes, course completion rates, and institutional performance — the kind of insights that help education businesses like Tetr, Unacademy, and BYJU'S make data-driven decisions.

---

## 🔍 Key Findings

| Insight | Finding |
|---|---|
| Test Prep Completion Rate | Only **33.5%** of students completed prep courses |
| Impact of Structured Learning | Students with prep scored **+7.7 points higher** on average |
| Pass Rate (All Subjects) | **80.5%** passed all three subjects |
| At-Risk Students | **12.4%** of students scoring below 50 average — need early intervention |
| High Performers | **21.1%** of students scoring 80+ average |

## 💡 Business Recommendations

1. **Boost course completion** — only 1 in 3 students complete structured prep; targeted nudges, reminders, and personalized outreach could significantly improve outcomes
2. **Early intervention for at-risk students** — 124 students scoring below 50 need personalized support before they drop off completely
3. **Socioeconomic factors impact performance** — students from disadvantaged backgrounds need additional support programs and counselling
4. **Structured learning ROI is clear** — +7.7 point improvement justifies investing in making prep courses more accessible and engaging globally
5. **Segment students by performance** — High, Medium, At-Risk, and Critical segments need different engagement strategies
